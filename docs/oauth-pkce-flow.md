# Authorization Code + PKCE 流程

```
第三方应用                           CP OAuth                         用户浏览器
    |                                  |                                  |
    |  1. 发起授权请求                   |                                  |
    |  (client_id, scope,              |                                  |
    |   redirect_uri, code_challenge)   |                                  |
    |--------------------------------->|                                  |
    |                                  |  2. 显示授权页面                  |
    |                                  |--------------------------------->|
    |                                  |                                  |
    |                                  |  3. 用户同意/拒绝                |
    |                                  |<---------------------------------|
    |                                  |                                  |
    |  4. 授权码回调                    |                                  |
    |  (code + state)                  |                                  |
    |<---------------------------------|                                  |
    |                                  |                                  |
    |  5. 换取 Token                   |                                  |
    |  (code + code_verifier +         |                                  |
    |   grant_type=authorization_code)  |                                  |
    |--------------------------------->|                                  |
    |                                  |                                  |
    |  6. 返回 Token                   |                                  |
    |  (access_token + refresh_token)  |                                  |
    |<---------------------------------|                                  |
    |                                  |                                  |
    |  7. 使用 Token 访问 UserInfo     |                                  |
    |  (Authorization: Bearer)          |                                  |
    |--------------------------------->|                                  |
    |                                  |                                  |
    |  8. 返回用户信息                 |                                  |
    |<---------------------------------|                                  |
```
